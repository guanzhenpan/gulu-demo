# PP UI 官网

PP UI 是一个实用的 UI 简易框架，基于Vue的UI组件库，提供了常用的组件，适合移动端和PC端使用。

# 支持

  • 🌞1.Button(按钮)
  • 🌞2.Input(输入框)
  • 🌞3.Grid(栅格)
  • 🌞4.Layout(布局)
  • 🌞5.Popover(弹出框)
  • 🌞6.Tabs(标签)
  • 🌞7.Toast(消息提示)
  • 🌞8.Collapse(折叠面板)
